# 🎓 Student Performance — Data Cleaning & EDA

## 📌 Overview
Cleaned and analyzed a messy real-world student performance 
dataset of 10,000+ rows using Python.

## 🛠️ Tools Used
- Python · Pandas · NumPy · Matplotlib · Seaborn

## 🧹 What I Did
- Handled missing values & removed duplicates
- Removed outliers using IQR method
- Renamed & dropped unnecessary columns
- Built Univariate, Bivariate & Trivariate visualizations
- Created Correlation Heatmap & Line plots
